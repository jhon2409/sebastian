¿Qué plataformas son compatibles con la Game Store?

La Game Store actualmente solo está disponible para PC y Mac. Puedes comprobar la compatibilidad de plataforma para títulos individuales consultando la sección "Acerca del juego" de cualquier página de producto.

¿Qué es eso de los juegos gratis?

Game Store ofrecerá un nuevo juego gratis disponible cada semana a lo largo de 2020. Cuando reclames un juego gratis, será tuyo para siempre, incluso después de que el juego ya no esté disponible para nuevos clientes de forma gratuita.
